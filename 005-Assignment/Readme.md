# Week 5 – Apache Spark DataFrames

## Objective

Understand Spark fundamentals and perform data cleaning, transformation, and aggregation using Spark DataFrames.

## Tasks Performed

- Studied the limitations of MapReduce and advantages of Spark.
- Learned Spark DataFrame concepts and immutability.
- Performed data cleaning operations such as removing duplicates and handling null values.
- Applied filtering conditions on datasets.
- Used aggregation functions (count, sum, avg, min, max).
- Grouped data using `groupBy()`.
- Understood wide transformations and shuffle operations.
- Modified DataFrame schema by casting and renaming columns.
- Built a simple Spark data processing pipeline.
